Python translation of a Fortran subroutine remesh for remeshing vortex
particles in a 2D grid. It is adapted from the code presented in:

- Magni, Adrien, and Georges-Henri Cottet. "Accurate, non-oscillatory,
remeshing schemes for particle methods." Journal of Computational
Physics 231.1 (2012): 152-172.  -
https://doi.org/10.1016/j.jcp.2011.09.005

- https://cse-lab.seas.harvard.edu/files/cse-lab/files/talks-paceth_2_remeshingmultiresolution.pdf
